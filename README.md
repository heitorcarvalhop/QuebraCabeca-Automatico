Jogo de quebra-cabeça em Python/Tkinter com:
 1.  Níveis de dificuldade (Fácil / Médio / Difícil)
 2.  Peças com formato real (encaixes nas bordas)
 3.  Arrastar peças manualmente
 4.  Botão de dica (mostra a posição correta de uma peça)
 5.  Temporizador
 6.  Contador de movimentos
 7.  Sistema de pontuação (tempo + dificuldade + movimentos)
 8.  Pré-visualização da imagem original
 9.  Sons e efeitos (encaixe, embaralhar, finalizar)
10.  Animação com aceleração/desaceleração e leve "overshoot"
11.  Salvar / carregar progresso
12.  Tela inicial (Novo jogo, Carregar imagem, Configurações, Ranking, Sair)
13.  Ranking de melhores tempos por dificuldade
14.  Modo automático passo a passo

Bibliotecas: tkinter, Pillow (PIL), random, math, json, wave, struct,
             tempfile, os, sys, subprocess
